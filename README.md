# Trivia-Quiz
Project Description
This is an interactive Trivia Quiz Web Application built using HTML, CSS, and JavaScript.
The application allows users to answer multiple-choice questions and receive instant feedback on their selections.
It demonstrates core front-end development concepts such as DOM manipulation, event handling, and dynamic UI updates.

# Features
Multiple-choice quiz questions
Instant feedback on answers
Score calculation at the end of the quiz
Interactive and user-friendly interface
Responsive design for smooth user experience

# Technologies Used
HTML – Structure of the application
CSS – Styling and layout
JavaScript – Quiz logic and interactivity
